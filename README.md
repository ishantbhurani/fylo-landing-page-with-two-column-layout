## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/fylo-landing-page-with-two-column-layout.git
```

Go to the project directory

```bash
  cd fylo-landing-page-with-two-column-layout
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
